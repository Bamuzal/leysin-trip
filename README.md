# Leysin Trip Planner

Static family trip dashboard for the June 20-30, 2026 Leysin, Switzerland trip.

## GitHub Pages

This site is designed to run directly from GitHub Pages. Publish the repository from the `main` branch root, then open the Pages URL.

https://bamuzal.github.io/leysin-trip/

The itinerary builder saves changes in each visitor's browser using localStorage. There is no shared backend or login.
